Проект с играми.

Для установки проекта выполните последовательно команды:

make install  
make build  
make package-install

Для вызова скрипта приветствия выполните команду:

make brain-games

Список доступных игр:

1. Игра: «Проверка на чётность».  
Для вызова игры введите команду:  
make brain-even
2. Игра: «Калькулятор»
Для вызова игры введите команду:  
make brain-calc
3. Игра «НОД»
Для вызова игры введите команду:  
make brain-gcd
4. Игра «Арифметическая прогрессия»
Для вызова игры введите команду:  
make brain-progression
5. Игра «Простое ли число?»
Для вызова игры введите команду:  
make brain-prime
